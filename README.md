# Safaisa
AI to vet award writeups for SAF
Only for commanders and clerks to use
Google sheet attached takes you to tracking list
Whenever an award is accepted by the pc for nomination it gets auto appended to the google sheet
not recommended to edit the google sheet unless neccessary 
--Using streamlit - website goes to sleep if left inactive for a while, hence I used - Uptime robot

UptimeRobot (FREE & Recommended)
What it does: Pings your app every 5 minutes to keep it awake
Setup (Takes 2 minutes):

Go to https://uptimerobot.com/
Create free account
Click "Add New Monitor"
Configure:

Monitor Type: HTTP(s)
Friendly Name: SAFAISA Award Vetter
URL: Your app URL (e.g., https://your-app.streamlit.app)
Monitoring Interval: 5 minutes


Click "Create Monitor"
